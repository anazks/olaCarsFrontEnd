## Data Flow

1. **Authentication:** User logs in via `AdminLogin`, receiving a JWT token stored in `localStorage`.
2. **Access Control:** `ProtectedRoute` validates the user's role from the JWT against the required roles for the route.
3. **API Interaction:** Components call service methods (e.g., `supplierService.getSuppliers()`), which use the shared `api` instance to make authorized HTTP requests.
4. **State Management:** Uses React Context (`ThemeContext`) for global UI states like theme-switching.
